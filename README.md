<h2 dir="rtl">✨ قابلیت‌های کلیدی</h2>

<p dir="rtl">
- <b>کلید فعال‌سازی اصلی:</b> روشن و خاموش کردن کل افزونه تنها با یک کلیک.<br>
- <b>تزریق فونت وزیر:</b> اعمال خودکار فونت باکیفیت و خوانای Vazirmatn روی تمام صفحات.<br>
- <b>راست‌چین هوشمند (RTL):</b> اصلاح جهت متون، فرم‌ها و جدول‌ها.<br>
- <b>ایزوله‌سازی کدهای برنامه‌نویسی:</b> با فعال کردن سوییچ مربوطه، بخش‌های کد (<code>code</code>، <code>pre</code>) دست‌نخورده و چپ‌چین باقی می‌مانند تا خوانایی کدهای فنی به هم نریزد.<br>
- <b>حفاظت از آیکون‌ها:</b> ممانعت هوشمند از تغییر فونت آیکون‌ها (مانند FontAwesome) برای جلوگیری از بهم‌ریختگی ظاهری سایت‌ها.
</p>
